Midi2Anglo

Midi2Anglo shows you fingering recommendations for Anglo concertina based on a midi file you upload. It's very much still a work in progress. Currently, you can load relatively simple midi files that only contain notes playable on a 30-button C/G instrument. More complicated files containing notes outside the concertina's playable range will not readable. We're working on it!

Fingering recommendations come from Owen Anderson's Concertina PBQP Solver. You can read more about how it works on concertina.net.

The front end is by Luke Hillman (see also: Anglo Piano).
